Statistics

Mean

Add all the values and divide by the amount of values.

Range

The difference between the smallest number and the biggest number.

Mode
  • The value that it's most repeated.
  • If there's no repeated number, there is no mode.
  • If there are two or more values that are repeated in the same amount, select all of them as the mode.
Median
  • Arrange all values from decreasing to increasing and check for the middle number.
  • In case there is no middle number, choose two numbers in the middle, add them and divide them by 2.
